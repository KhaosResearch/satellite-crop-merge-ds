# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-24

### Added
- Initial release of the Satellite Crop & Merge Data Space service.
- Gradio interface for satellite data retrieval.
- Integration with KHAOS' MinIO database and Copernicus/Sentinel API.
- Support for GeoJSON parcel geometry upload.
- Support for SIGPAC cadastral reference (Spain).
- Interactive map for manual area delimitation.
- Apache 2.0 License.
- Comprehensive documentation in the `docs/` directory.

### Changed
- Refactored repository documentation to improve clarity and maintenance.
- Migrated technical details from root README to specialized documentation files.
